# Eventos de Mouse 

Exercício extra para o curso de Javascript do Curso em Vídeo do professor Gustavo Guanabara. O exercício tem como objetivo treinar os **eventos DOM**, mais específicamente os eventos de mouse **mouseenter**, **mouseleave**, **mousedown** e **mouseup**. Além disso, o exercício pratica a **manipulação de elementos HTML** e a criação de **Funções**.


Nele, temos uma div de 250 X 250 px, na cor cinza. Quando botamos o mouse em cima dessa div, ela se torna amarela, e o texto "Bem-Vindo!" aparece no meio da div. Quando o mouse sai da div, ela retorna para a cor cinza, e o texto "Passe o mouse" aparece. Quando o usuário clica em cima da div com o botão esquerdo do mouse, ela se torna vermelha, e o texto "Segurando!" aparece. Ao soltar o botão esquerdo do mouse, a div fica amarela, e o texto "Soltou!" aparece.


# Mouse Events


Extra exercise for the Javascript course from Curso em Vídeo by Professor Gustavo Guanabara. The exercise aims to practice **DOM events**, specifically the mouse events **mouseenter**, **mouseleave**, **mousedown**, and **mouseup**. Additionally, the exercise practices **manipulating HTML elements** and creating **Functions**.

In it, we have a 250 x 250 px gray div. When we hover the mouse over this div, it turns yellow, and the text "Bem-Vindo!" appears in the middle of the div. When the mouse leaves the div, it returns to gray, and the text "Passe o mouse" appears. When the user clicks on the div with the left mouse button, it turns red, and the text "Segurando!" appears. Upon releasing the left mouse button, the div turns yellow, and the text "Soltou!" appears.
